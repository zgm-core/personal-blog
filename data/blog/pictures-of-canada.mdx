加拿大风景如画，有枫叶、雪山、碧绿湖泊和多伦多。在这个图片展览中欣赏美景，看看如何用 MDX 和 Tailwind CSS 轻松复制这种效果。

图片使用 `next/image` 组件加载。本地存储的图片位于以下路径：`/static/images/canada/[文件名].jpg`

由于使用的是 mdx，我们可以创建一个简单的响应式 flexbox 网格来展示图片，只需几个 tailwind css 类即可。
