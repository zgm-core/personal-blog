# 什么是 Markdown？

Markdown 是一种在网页上格式化文本的方式。你可以控制文档的显示效果；将文字加粗或斜体、添加图片、创建列表等，只需要几个非字母字符就能实现，比如 `#` 或 `*`。

## 表格

你可以通过组合单词列表来创建表格，用连字符 `-` 分隔第一行，然后用竖线 `|` 分隔每个列：

```
| 第一个表头 | 第二个表头 |
| --------- | --------- |
| 单元格 1 的内容 | 单元格 2 的内容 |
| 第一列的内容 | 第二列的内容 |
```

| 第一个表头      | 第二个表头      |
| --------------- | --------------- |
| 单元格 1 的内容 | 单元格 2 的内容 |
| 第一列的内容    | 第二列的内容    |

## 删除线

任何用两个波浪号包裹的单词（如 `~~这样~~`）会显示为~~删除线效果~~。
