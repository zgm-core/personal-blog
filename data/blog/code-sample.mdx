## 代码块

JavaScript 代码示例：

```javascript
var num1, num2, sum
num1 = prompt('请输入第一个数字')
num2 = prompt('请输入第二个数字')
sum = parseInt(num1) + parseInt(num2) // "+" 表示相加
alert('总和 = ' + sum) // "+" 表示拼接成字符串
```

Python 代码示例：

```python
def fib():
    a, b = 0, 1
    while True:            # 第一次迭代：
        yield a            # 先产出 0，然后
        a, b = b, a + b    # a 变成 1，b 也变成 1 (0 + 1)

for index, fibonacci_number in zip(range(10), fib()):
     print('{i:3}: {f:3}'.format(i=index, f=fibonacci_number))
```
