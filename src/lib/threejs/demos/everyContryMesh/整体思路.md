### 国家球面 Mesh 生成思路

1. 多边形轮廓内生成一系列等间距点阵
2. 对点集进行三角剖分，生成国家平面 Mesh
3. 国家平面 Mesh 转球面 Mesh：国家平面 Mesh 三角形顶点经纬度坐标转球面坐标即可

### delaunator 库(三角剖分)

github 地址：https://github.com/mapbox/delaunator

```javascript
npm install delaunator -S
```

```javascript
<script src="https://unpkg.com/delaunator@4.0.1/delaunator.js"></script>;

Delaunator.from().triangles;
```

```javascript
import Delaunator from "delaunator";

Delaunator.from().triangles;
```
