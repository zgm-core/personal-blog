## 概述

本示例将每个国家创建为独立的 3D Mesh：

- 从 GeoJSON 多边形数据生成国家 Mesh
- 每个国家是独立的可交互对象
- 支持鼠标悬停高亮
- 支持点击选中某个国家
- 可对选中国家进行颜色、透明度变换

这是实现国家级别交互式地图的基础技术。
