## 概述

本示例展示 Three.js 中贝塞尔曲线的使用：

- 使用 `CubicBezierCurve3` 创建三次贝塞尔曲线
- 使用 `QuadraticBezierCurve3` 创建二次贝塞尔曲线
- 控制点调整曲线形状
- 曲线可视化渲染

贝塞尔曲线是飞线效果的基础，后续的 3D 飞线、OA 线等都基于此技术。
