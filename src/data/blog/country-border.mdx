## 概述

本示例演示如何使用 Three.js 在地球球体上绘制国家边界：

- 加载 GeoJSON 格式的国家边界数据
- 将经纬度坐标转换为 3D 球面坐标
- 使用 Line 或 LineSegments 绘制边界线
- 不同国家使用不同颜色区分

这是地球可视化系列的基础示例，后续的线框、轮廓线等效果都基于此实现。
