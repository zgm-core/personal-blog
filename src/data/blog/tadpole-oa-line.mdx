## 概述

本示例在 OA 飞线基础上添加蝌蚪状拖尾：

- 飞线头部有明亮的"头部"光点
- 尾部逐渐变细、变暗，形成蝌蚪形状
- 使用 shader 实现渐变透明度
- 拖尾长度和衰减速度可调节

蝌蚪状飞线比等宽飞线更具动感和方向性，常用于攻击溯源、数据流向等场景。
