## 概述

本示例展示如何在地球球面上添加标注：

- 使用 CSS2DRenderer 或 Sprite 在球面上放置标签
- 标注国家名称、城市名等文字信息
- 标签始终面向相机（Billboard 效果）
- 根据缩放级别显示/隐藏标签

球面标注是地理信息系统中常见的需求，用于标识地名和关键位置。
