## 概述

本示例是一个综合性项目实战：

- 整合国家边界、轮廓线、Mesh 渲染
- 添加飞线、光柱、标注等视觉效果
- 实现数据驱动的颜色映射（人口密度等）
- 完整的交互功能：缩放、旋转、点击、悬停
- 适合作为数据大屏项目的参考模板

这个案例展示了如何将各种 Three.js 技术组合成完整的产品。
