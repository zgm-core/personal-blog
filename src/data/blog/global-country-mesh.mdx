## 概述

本示例渲染完整的全球国家 Mesh：

- 一次性加载全球所有国家的 GeoJSON 数据
- 每个国家生成对应的 3D Mesh 贴附在球面上
- 统一管理所有国家 Mesh 的渲染
- 保持性能的同时支持大量多边形渲染

全球国家 Mesh 是实现世界地图交互的完整方案。
